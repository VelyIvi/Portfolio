AI learns to drive

Coderclass: 2022-2023 first semester

A reinforcement learning system created from the ground up

The goal of this project was to make an artificial intelligence (AI) system made from scratch that could complete a track, without relying on AI libraries. The AI-system uses reinforcement learning to update its neural network (its brain). Reinforcement learning does this sequentially by altering values of the parent a lot of times and putting each variation in to the simulation. The actor that is the best at completing the circuit is chosen to be the parent of the next iteration. After a lot of iterations there will be a neural network that can drive the track perfectly.

Furthermore, the program is coded in C/C++ and uses Raylib as its graphics engine. This choice is motivated by the fact that I wanted to use the skills I've learned the year before in the VU C++ course.

Introduction to Godot

Coderclass: 2021-2022 second semester

Introduction to Godot: an open source game engine

Godot is a lightweight open source game engine. The reason I chose to try out Godot was that Unity didn't work well on linux where I migrated to for better compatibility with C++. The main difference of Godot with Unity is the way that nodes/objects are coded. Unity uses components for their objects, with each object having the ability to have multiple components and scripts, while Godot uses nodes which have a single purpose e.g. managing the physics or another node managing the collision. By having a node only complete one assignment makes the project clearer to navigate, but I also find this a double-edged sword since at the time the code could become very long with multiple purposes.

The idea was to make a small 2d platformer shooter. The game was not finished due to that the project taking place during a turbulent time for myself. However, I still have a desire to complete at least one project in Godot since I did like using the engine. And with recent updates and improvements to the engine I feel like I am able to make my dream a reality.

Coderclass

Metis: 2018-2021

A class for secondary school students to learn the basics of coding

The Coder class is a profile at the Metis in which computer science is an important part. Particularly motivated students with a great interest in ICT are part of the target group. From year 1 onwards, students receive an intensive program of 5 hours a week in which they learn the tricks of the trade. Every student can work at his own pace to deepen or broaden his knowledge. The students also do a project four times a year that involves a real client (company or further education). What is special is that the MML collaborates intensively with companies. Teachers and students can do an internship there. In addition, various companies provide the school with realistic assignments, so that students really know how the business world works.
